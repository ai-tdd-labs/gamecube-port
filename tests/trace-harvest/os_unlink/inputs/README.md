# OSUnlink trigger inputs

Place deterministic Dolphin movie files (`.dtm`) here for RVZ trigger replay.

Default file used by tooling:

- `mp4_overlay_trigger_001.dtm`

Command:

- `tools/harvest_and_replay_os_unlink_movie.sh`

The script auto-loads RVZ from `docs/sdk/mp4/MP4_assets.md` and this default
DTM path when no arguments are provided.
